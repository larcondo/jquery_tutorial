jQuery Traversing

¿Qué es Traversing?

jQuery traversing, que significa "moverse a través", es utilizado para "encontrar" (o seleccionar) elementos HTML basándose en su relación con otros elementos. Se comienza con una selección y se mueve a través de esa selección hasta alcanzar el elemento deseado.

La imagen siguiente ilustra una página HTML como un árbol (el árbol del DOM). Con jQuery traversing, podemos fácilmente movernos hacia arriba (ancestros), hacia abajo (descendientes) y hacia los costados (hermanos) en el árbol, comenzando por el elemento seleccionado (el elemento actual). Este movimiento es llamado traversing - o moverse a través - del árbol del DOM.

Explicación de la ilustración:

  • El elemento <div> es el padre del <ul> y un ancestro de todo lo que hay dentro de él.
  • El elemento <ul> es el padre de ambos elementos <li> y un hijo del <div>
  • El elemento <li> izquierdo es el padre de los <span>, hijo del <ul> y un descendiente del <div>
  • El elemento <span> es un hijo del <li> izquierdo y un descendiente de <ul> y <div>
  • Los dos elementos <li> son hermanos (comparten el mismo padre)
  • El elemento <li> derecho es el padre de <b>, hijo de <ul> y un descendiente de <div>
  • El elemento <b> es un hijo del <li> derecho y un descendiente de <ul> y <div>
    • Un ancestro es un padre, un abuelo, un bis-abuelo, y así sucesivamente. Un descendiente es un hijo, un nieto, un bis-nieto, y así sucesivamente. Los hermanos comparten el mismo padre.
