jQuery Traversing - Ancestros

Con jQuery podemos recorrer hacia arriba el árbol DOM para encontrar ancestros de un elemento.

Un ancestro es un padre, un abuelo, un bis-abuelo, y así sucesivamente.

Recorrer hacia arriba el árbol DOM

Tres métodos jQuery útiles para recorrer hacia arriba el árbol DOM son:

  • parent()
  • parents()
  • parentsUntil()

    • El método jQuery parent()

    El método parent() retorna el elemento padre directo del elemento seleccionado.

    Este método solo recorre un solo nivel hacia arriba en el árbol DOM.

    El siguiente ejemplo retorna el elemento padre directo de cada elemento <span>:

    Ejemplo

    
  $(document).ready(function(){
    $("span").parent();
  });

    El método jQuery parents()

    El método parents() retorna todos los elementos ancestros del elemento seleccionado, recorriendo todo hacia arriba hasta el elemento raíz del documento (<html>).

    El siguiente ejemplo retorna todos los ancestros de todos los elementos <span>:

    Ejemplo

    
  $(document).ready(function(){
    $("span").parents();
  });

    También podemos usar un parámetro opcional para filtrar la búsqueda de los ancestros.

    El siguiente ejemplo retorna todos los ancestros de todos los elementos <span> que son elementos <ul>:

    Ejemplo

    
  $(document).ready(function(){
    $("span").parents("ul");
  });

    El método jQuery parentsUntil()

    El método parentsUntil() retorna todos los elementos ancestros entre dos argumentos dados.

    El ejemplo siguiente retorna todos los elementos ancestros entre un elemento <span> y un <div>:

    Ejemplo

    
  $(document).ready(function(){
    $("span").parentsUntil("div");
  });
