jQuery Stop Animations

El método jQuery stop()

El método jQuery stop() es utilizado para para animaciones o efectos antes de que finalicen.

El método stop() funciona para todas las funciones de efectos jQuery, incluyendo las animacions de desplazamieto (sliding), desvanecimiento (fading) y las animaciones personalizadas.

Sintaxis:


  $(selector).stop(stopAll,goToEnd);

El parámetro opcional stopAll especifica si la cola de animaciones debe ser borrada o no. Por defecto su valor es false, lo que significa que solo la animación activa se debe parar, permitiendo que las animaciones siguientes en la cola se ejecuten luego.

El parámetro opcional goToEnd especifica si completar la animación actual inmediatamente o no. Por defecto su valor es false.

Así que, por defecto, el método stop() "mata" la animación actual que está siendo ejecutada en el elemento seleccionado.

El siguiente ejemplo muestra el uso del método stop(), sin ningún parámetro:


  $("#stop").click(function(){
    $("#panel").stop();
  });
