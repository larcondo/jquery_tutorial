Efectos jQuery - Animation

jQuery Animations - El método animate()

El método animate() es utilizado para crear animaciones personalizadas.

Sintaxis:


  $(selector).animate({params},speed,callback);

El parámetro obligatorio params define las propiedades CSS a ser animadas.

El parámetro opcional speed define la duración del efecto. Puede tener los siguientes valores: "slow", "fast" o milisegundos.

El parámetro opcional callback es una función a ser ejecutada después que la animación finalice.

El siguiente ejemplo muestra un simple uso del método animate(), donde se mueve el elemento <span> hacia la derecha hasta que alcance 250px en la propiedad "left":


  $("button").click(function(){
    $("div").animate({left: '250px'});
  });
Por defecto, todos los elementos HTML tienen una posición estática y no se pueden mover. Para manipular la posición, recuerda primero establecer la propiedad CSS "position" del elemento a "relative", "fixed" o "absolute"!

jQuery animate() - Manipular múltiples propiedades

Notemos que multiples propiedades pueden ser animadas al mismo tiempo:


  $("button").click(function(){
    $("div").animate({
      left: '250px',
      opacity: '0.5',
      height: '150px',
      width: '150px'
    });
  });
¿Es posible manipular TODAS las pripiedades CSS con el método animate()?
Si, casi! Sin embargo, hay una cosa importante a recordar: todos los nombres de las propiedades deben tener nomenclatura "camel-cased" cuando son utilizadas con el método animate(). Debemos escribir paddingLeft en vez de padding-left, marginRight en vez de margin-right, y así sucesivamente.

La animación de color no está incluída en la librería principal jQuery. Si queremos una animación de color, necesitaremos descargar el Color Animations plugin desde jQuery.com.

jQuery animate() - Usar valores relativos

También es posible definir valores relativos (relativo al valor actual del elemento). Esto se hace poniendo += o -= antes del valor:


  $("button").click(function(){
    $("div").animate({
      left: '250px',
      height: '+=150px',
      width: '+=150px'
    });
  });

jQuery animate() - Usar valores pre-definidos

Incluso podemos especificar un valor de animación de la propiedad como "show", "hide" o "toggle":


  $("button").click(function(){
    $("div").animate({
      height: 'toggle'
    });
  });

jQuery animate() - Usar la funcionalidad de cola

Por defecto, jQuery viene con una funcionalidad de cola para las animaciones.

Esto quiere decir que si escribimos multiples llamadas a animation(), una después de otra, jQuery crea una cola "interna" con esas llamadas al método. Luego, ejecuta esas llamadas una por una.

Entonces, si queremos realizar diferentes animaciones una despés de otra, podemos hacer uso de esta funcionalidad:


  $("button").click(function(){
    var div = $("div");
    div.animate({height: '300px', opacity: '0.4'}, "slow");
    div.animate({width: '300px', opacity: '0.8'}, "slow");
    div.animate({height: '100px', opacity: '0.4'}, "slow");
    div.animate({width: '100px', opacity: '0.8'}, "slow");
  });

El ejemplo siguiente mueve el elemento <div> hacia la derecha y luego incrementa el tamaño del texto:


  $("button").click(function(){
    var div = $("div");
    div.animate({left: '100px'}, "slow");
    div.animate({fontSize: '3em'}, "slow");
  });
