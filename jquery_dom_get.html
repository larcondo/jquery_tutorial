jQuery - Obtener (Get) Contenido y Atributos

jQuery tiene poderosos métodos para encadenar y manipular elementos y atributos HTML.

Manipulación del DOM jQuery

Una parte muy importante de jQuery es la posibilidad de manipular el DOM.

jQuery viene con un grupo de métodos relativos al DOM que hacen más fácil el acceso y manipulación de los elementos y atributos.

DOM = Document Object Model
El DOM define un estándar para el acceso a documentos HTML y XML:
"El W3C Document Object Model (DOM) es una plataforma y lenguaje neutral que permite a programas y scripts acceder y actualizar dinamicamente el contenido, estructura y estilo de un documento.

Contenido Get - text(), html() y val()

Tres simples, pero útiles, métodos jQuery para manipular el DOM son:

  • text() - Establece o retorna el contenido de texto de los elementos seleccionados
  • html() - Establece o retorna el contenido de los elementos seleccionados (incluído el markup HTML)
  • val() - Establece o retorna el valor de los campos de formulario

    • El ejemplo siguiente muestra como obtener el contenido con los métodos jQuery text() y html():

    Ejemplo

    
  $("#btn1").click(function(){
    alert("Text: " + $("#test").text());
  });
  $("#btn2").click(function(){
    alert("HTML: " + $("#test").html());
  });

    El ejemplo siguiente muestra como obtener el valor de un campo "input" con el método jQuery val():

    Ejemplo

    
  $("#btn1").click(function(){
    alert("Value: " + $("#test").val());
  });

    Atributos Get - attr()

    El método jQuery attr() es utilizado para obtener (get) valores de atributos.

    El ejemplo siguiente muestra como obtener el valor del atributo "href" en un elemento link:

    Ejemplo

    
  $("button").click(function(){
    alert($("#w3s").attr("href"));
  });
