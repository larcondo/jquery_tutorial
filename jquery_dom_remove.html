jQuery - Quiter Elementos (Remove)

Con jQuery es fácil quitar elementos HTML existentes.

Quitar Elementos/Contenido

Para quitar elementos y contenido hay dos principales métodos jQuery:

  • remove() - Quita el elemento seleccionado (y sus elementos hijos)
  • empty() - Quita los elementos hijo del elemento seleccionado

    • El método jQuery remove()

    El método jQuery remove() quita el elemento seleccionado y sus elementos hijos.

    Ejemplo

    
  $("#div1").remove();

    El método jQuery empty()

    El método jQuery empty() quita los elementos hijo del elemento seleccionado.

    Ejemplo

    
  $("#div1").empty();

    Filtrar los Elementos ha ser removidos

    El método jQuery remove() también acepta un parámetro el cual nos permite filtrar los elementos a ser quitados.

    El parámetro puede ser cualquiera de los selectores jQuery.

    El ejemplo siguiente quita todos los elementos <p> con class="test":

    Ejemplo

    
  $("p").remove(".test");

    El ejemplo que sigue quita todos los elementos <p> con class="test" y class="demo":

    Ejemplo

    
  $("p").remove(".test, .demo");
