jQuery - Añair Elementos

Con jQuery es fácil añadir nuevos elementos/contenido.

Añadir Contenido HTML Nuevo

Veremos cuatro métodos jQuery que son utilizados para añadir contenido:

  • append() - Inserta contenido al final de los elementos seleccionados
  • prepend() - Inserta contenido al principio de los elementos seleccionados
  • after() - Inserta contenido después de los elementos seleccionados
  • before() - Inserta contenido antes de los elementos seleccionados

    • El método jQuery append()

    El método jQuery append() inserta contenido AL FINAL de los elementos HTML seleccionados.

    Ejemplo

    
  $("p").append("Some appended text.");

    El método jQuery prepend()

    El método jQuery prepend() inserta contenido AL COMIENZO de los elementos HTML seleccionados.

    Ejemplo

    
  $("p").prepend("Some prepended text.");

    Añadir Varios Elementos Nuevos con append() y prepend()

    En ambos ejemplos anteriores solo hemos insertado algún texto al comienzo o al final de los elementos HTML seleccionados.

    Sin embargo, ambos métodos append() y prepend() pueden tomar un número infinito de nuevos elementos como parámetros. Los nuevos elementos pueden ser generados con texto/HTML, con jQuery o con código JavaScript y elementos del DOM.

    En el ejemplo siguiente, crearemos varios nuevos elementos. Los elementos son creados con texto/HTML, jQuery y JavaScript/DOM. Luego añadiremos los nuevos elementos al texto con el método append() (también funcionaría para prepend()):

    Ejemplo

    
  function appendText() {
    var txt1 = "<p>Text.</p>";               // Create element with HTML 
    var txt2 = $("<p></p>").text("Text.");   // Create with jQuery
    var txt3 = document.createElement("p");  // Create with DOM
    txt3.innerHTML = "Text.";
    $("body").append(txt1, txt2, txt3);      // Append the new elements
  }

    Los métodos jQuery after() y before()

    El método jQuery after() inserta contenido DESPUÉS de los elementos seleccionados.

    El método jQuery before() inserta contenido ANTES de los elementos seleccionados.

    Ejemplo

    
  $("img").after("Some text after");

  $("img").before("Some text before");

    Añadir Varios Elementos Nuevos con after() y before()

    También, ambos métodos after() y before() pueden tomar un infinito número de elementos como parámetros. Los nuevos elementos pueden ser generados con texto/HTML, con jQuery o con código JavaScript.

    En el ejemplo siguiente crearemos varios nuevos elementos. Los elementos son creados con texto/HTML, jQuery y JavaScript/DOM. Luego insertaremos los nuevos elementos al texto con el método after() (también funcionaría para before):

    Ejemplo

    
  function afterText() {
    var txt1 = "<b>I </b>";                    // Create element with HTML 
    var txt2 = $("<i></i>").text("love ");     // Create with jQuery
    var txt3 = document.createElement("b");    // Create with DOM
    txt3.innerHTML = "jQuery!";
    $("img").after(txt1, txt2, txt3);          // Insert new elements after <img>
  }
