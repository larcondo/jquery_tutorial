Tutorial jQuery

Introducción

jQuery es una libería JavaScript.

jQuery simplifica mucho la programacion JavaScript y es fácil de aprender.

El propósito de jQuery es hacer mucho más fácil el uso de JavaScript en un sitio web.

jQuery toma un montón de tareas comunes que requieren muchas líneas de código JavaScript y las reúne en métodos que pueden ser llamados con una sola línea de código.

jQuery también simplifica muchas cosas complicadas de JavaScript como AJAX y la manipulación del DOM.

La librería jQuery contiene las siguientes características:

  • Manipulación HTML/DOM
  • Manipulación CSS
  • Métodos de eventos HTML
  • Efectos y animaciones
  • AJAX
  • Utilidades (Utilities)

    • Añadir jQuery a tus páginas web

    Hay varias maneras de comenzar a utilizar jQuery en tu sitio web. Puedes:

  • Descargar la libería jQuery de jQuery.com
  • Incluir jQuery desde un CDN, como Google

    • Descargar jQuery

    Hay dos versiones disponibles de jQuery para descargar:

  • Product version
  • Development version

    • Ambas versiones se pueden descargar de jQuery.com

    La librería jQuery es un solo archivo JavaScript y se puede agregar con el tag HTML <script>:

    
  <head>
    <script src="jquery-3.6.1.min.js">
  </head>

    jQuery CDN

    Si no quieres descargar jQuery, lo puedes incluir desde un CDN (Content Delivery Network).

    Google es un ejemplo de alguien que hostea jQuery:

    
  <head>
    <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.6.1/jquery.min.js">
  </head>
