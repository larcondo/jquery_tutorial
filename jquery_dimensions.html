jQuery - Dimensiones

Con jQuery es fácil trabajar con las dimensiones de los elementos y la ventana del navegador.

Métodos jQuery de Dimensión

jQuery tiene varios métodos importantes para trabajar con dimensiones:

  • width()
  • height()
  • innerWidth()
  • innerHeight()
  • outerWidth()
  • outerHeight()

    • Dimensinones jQuery

    Los métodos jQuery width() y height()

    El método width() establece o retorna el ancho de un elemento (excluyendo el padding, border y margin).

    El método height() establece o retorna el alto de un elemento (excluyendo el padding, border y margin).

    El siguiente ejemplo retorna el ancho y el alto de un elemento <div> específico:

    Ejemplo

    
  $("button").click(function(){
    var txt = "";
    txt += "Width: " + $("#div1").width() + "</br>";
    txt += "Height: " + $("#div1").height();
    $("#div1").html(txt);
  });

    Los métodos jQuery innerWidth() e innerHeight()

    El método innerWidth() retorna el ancho de un elemento incluyendo el padding.

    El método innerHeight() retorna el alto de un elemento incluyendo el padding.

    El ejemplo siguiente retorna el alto/ancho interior de un elemento <div> específico:

    Ejemplo

    
  $("button").click(function(){
    var txt = "";
    txt += "Inner width: " + $("#div1").innerWidth() + "</br>";
    txt += "Inner height: " + $("#div1").innerHeight();
    $("#div1").html(txt);
  });

    Los métodos jQuery outerWidth() e outerHeight()

    El método outerWidth() retorna el ancho de un elemento incluyendo el padding y el border.

    El método outerHeight() retorna el alto de un elemento incluyendo el padding y el border.

    El siguiente ejemplo retorna el ancho/alto externo de un elemento <div> específico:

    Ejemplo

    
  $("button").click(function(){
    var txt = "";
    txt += "Outer width: " + $("#div1").outerWidth() + "</br>";
    txt += "Outer height: " + $("#div1").outerHeight();
    $("#div1").html(txt);
  });

    El método outerWidth(true) retorna el ancho de un elemento incluyendo el padding, el border y el margin.

    El método outerWidth(true) retorna el alto de un elemento incluyendo el padding, el border y el margin.

    Ejemplo

    
  $("button").click(function(){
    var txt = "";
    txt += "Outer width (+margin): " + $("#div1").outerWidth(true) + "</br>";
    txt += "Outer height (+margin): " + $("#div1").outerHeight(true);
    $("#div1").html(txt);
  });

    width() y height() en document y window

    El siguiente ejemplo retorna el ancho y el alto de un documento (el documento HTML) y la ventana (el viewport del navegador):

    Ejemplo

    
  $("button").click(function(){
    var txt = "";
    txt += "Document width/height: " + $(document).width();
    txt += "x" + $(document).height() + "\n";
    txt += "Window width/height: " + $(window).width();
    txt += "x" + $(window).height();
    alert(txt);
  });

    El ejemplo siguiente establece el ancho y el alto de un elemento <div> específico:

    Ejemplo

    
  $("button").click(function(){
    $("#div1").width(500).height(500);
  });
