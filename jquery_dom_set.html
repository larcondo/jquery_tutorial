jQuery - Establecer (Set) Contenido y Atributos

Establecer Contenido - text(), html() y val()

Usaremos los mismos tres métodos de la sección anterior para establecer el contenido:

  • text() - Establece o retorna el contenido de texto de los elementos seleccionados
  • html() - Establece o retorna el contenido de los elementos seleccionados (incluído el markup HTML)
  • val() - Establece o retorna el valor de los campos de formulario

    • El ejemplo siguiente muestra como establecer el contenido con los métodos jQuery text(), html() y val():

    Ejemplo

    
  $("#btn1").click(function(){
    $("#test1").text("Hello world!");
  });
  $("#btn2").click(function(){
    $("#test2").html("<b>Hello world!</b>");
  });
  $("#btn3").click(function(){
    $("#test3").val("Dolly Duck");
  });

    Función Callback para text(), html() y val()

    Todos los métodos anteriores text(), html() y val() también vienen con una función de callback. La función callback tiene dos parámetros: el índice del elemento actual en la lista de elementos seleccionados y el valor original (anterior). Entonces podremos retornar el string que querramos utilizar como el nuevo valor desde la función.

    El siguiente ejemplo muestra los métodos text() y html() con una función callback:

    Ejemplo

    
  $("#btn1").click(function(){
    $("#test1").text(function(i, origText){
      return "Old text: " + origText + " New text: Hello world!
      (index: " + i + ")";
    });
  });
  
  $("#btn2").click(function(){
    $("#test2").html(function(i, origText){
      return "Old html: " + origText + " New html: Hello world!
      (index: " + i + ")";
    });
  });

    Setear Atributos - attr()

    El método jQuery attr() también se utiliza para establecer/cambiar valores de atributos.

    El ejemplo siguiente muestra cómo cambiar (set) el valor del atributo "href" de un elemento link:

    Ejemplo

    
  $("button").click(function(){
    $("#w3s").attr("href", "https://www.w3schools.com/jquery/");
  });

    El método attr() también nos permite establecer múltiples attributos al mismo tiempo.

    El siguiente ejemplo muestra cómo establecer los atributos "href" y "title" al mismo tiempo:

    Ejemplo

    
  $("button").click(function(){
    $("#w3s").attr({
      "href" : "https://www.w3schools.com/jquery/",
      "title" : "W3Schools jQuery Tutorial"
    });
  });

    Función Callback para attr()

    El método jQuery attr() también viene con una función callback. Dicha función callback tiene dos parámetros: el índice del elemento actual en la lista de los elementos seleccionados y el valor original (anterior) del atributo. Así, podremos retornar el string que querramos utilizar como nuevo valor de atributo desde la función.

    El siguiente ejemplo muestra el método attr() con una función callback:

    Ejemplo

    
  $("button").click(function(){
    $("#w3s").attr("href", function(i, origValue){
      return origValue + "/jquery/";
    });
  });
