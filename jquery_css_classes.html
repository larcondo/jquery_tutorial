jQuery - Get y Set clases CSS

Con jQuery es fácil manipular el estilo de los elementos.

Manipular CSS con jQuery

jQuery tiene varios métodos para manipular CSS. Entre dichos métodos veremos los siguientes:

  • addClass() - Añade una o más clases a los elementos seleccionados
  • removeClass() - Quita una o más clases a los elementos seleccionados
  • toggleClass() - Alterna entre agregar/quitar clases de los elementos seleccionados
  • css() - Establece o retorna el atributo "style"

    • Hoja de Estilo Ejemplo

    La siguiente hoja de estilos será utilizada para todos los ejemplos en este capítulo:

    
  .important {
    font-weight: bold;
    font-size: xx-large;
  }

  .blue {
    color: blue;
  }

    El método jQuery addClass()

    El siguiente ejemplo muestra cómo añadir atributos "class" a diferentes elementos. Además, podemos seleccionar múltiples elementos cuando añadimos clases:

    Ejemplo

    
  $("button").click(function(){
    $("h1, h2, p").addClass("blue");
    $("div").addClass("important");
  });

    Podemos, también, especificar múltiples clases dentro del método addClass():

    Ejemplo

    
  $("button").click(function(){
    $("#div1").addClass("important blue");
  });

    El método jQuery removeClass()

    El ejemplo siguiente muestra cómo quitar un atributo "class" específico de diferentes elementos:

    Ejemplo

    
  $("button").click(function(){
    $("h1, h2, p").removeClass("blue");
  });

    El método jQuery toggleClass()

    El siguiente ejemplo muestra cómo utilizar el método jQuery toggleClass(). Este método alterna entre añadir/quitar clases en los elementos seleccionados:

    Ejemplo

    
  $("button").click(function(){
    $("h1, h2, p").toggleClass("blue");
  });
