Efectos jQuery - Sliding

Métodos jQuery de deslizamiento (Sliding)

Con jQuery podemos crear efectos de deslizamiento en los elementos.

Para ello, jQuery tiene los siguientes métodos:

  • slideDown()
  • slideUp()
  • slideToggle()

    • El método jQuery slideDown()

    El método slideDown() es utilizado para deslizar hacia abajo un elemento.

    Sintaxis:

    
  $(selector).slideDown(speed,callback);

    El parámetro opcional speed especifica la duración del efecto. Puede tomar los siguientes valores: "slow", "fast" o milisegundos.

    El parámetro opcional callback es una función a ser ejecutada luego de que el deslizamiento finalice.

    El ejemplo siguiente muestra el método slideDown():

    
  $("#flip").click(function(){
    $("#panel").slideDown();
  });

    El método jQuery slideUp()

    El método slideUp() es utilizado para deslizar hacia arriba un elemento.

    Sintaxis:

    
  $(selector).slideUp(speed,callback);

    El parámetro opcional speed especifica la duración del efecto. Puede tomar los siguientes valores: "slow", "fast" o milisegundos.

    El parámetro opcional callback es una función a ser ejecutada luego de que el deslizamiento finalice.

    El ejemplo siguiente muestra el método slideUp():

    
  $("#flip").click(function(){
    $("#panel").slideUp();
  });

    El método jQuery slideToggle()

    El método slideToggle() alterna entre los métodos slideDown() y slideUp().

    Si el elemento ha sido deslizado hacia abajo, slideToggle() lo deslizará hacia arriba.

    Si el elemento ha sido deslizado hacia arriba, slideToggle() lo deslizará hacia abajo.

    Sintaxis:

    
  $(selector).slideToggle(speed,callback);

    El parámetro opcional speed puede tomar los siguientes valores: "slow", "fast" o milisegundos.

    El parámetro opcional callback es una función a ser ejecutada luego de que el deslizamiento finalice.

    El ejemplo siguiente muestra el método slideToggle():

    
  $("#flip").click(function(){
    $("#panel").slideToggle();
  });
