jQuery - El método css()

El método css() establece o retorna una o más propiedades de estilo para los elementos seleccionados.

Retornar una Propiedad CSS

Para retornar el valor específico de una propiedad CSS, utilizamos la siguiente sintaxis:


  css("propertyname");

El siguiente ejemplo retornará el valor del "background-color" del primer elemento que coincida con el elemento seleccionado:

Ejemplo


  $("p").css("background-color");

Establecer una Propiedad CSS

Para setear una propiedad CSS específica, utilizaremos la siguiente sintaxis:


  css("propertyname","value");

El siguiente ejemplo seteará el valor del "background-color" de TODOS los elementos que coincidan con la selección:

Ejemplo


  $("p").css("background-color", "yellow");

Establecer Múltiples Propiedades CSS

Para setear múltiples propiedades CSS, utilizamos la siguiente sintaxis:


  css({"propertyname":"value","propertyname":"value",...});

El siguiente ejemplo seteará un "background-color" y un "font-size" para todos los elementos que coinciden con la selección:

Ejemplo


  $("p").css({"background-color": "yellow", "font-size": "200%"});
