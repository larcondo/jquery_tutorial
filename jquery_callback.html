jQuery Callback Functions

Una función de callback es ejecutada luego de que el efecto actual esté 100% finalizado.

Funciones Callback jQuery

Las declaraciones (statements) de JavaScript son ejecutadas línea por línea. Sin embargo, con los efectos, la siguiente línea de código puede ser ejecutada incluso si el efecto no ha finalizado. Esto puede crear errores.

Para prevenir esto podemos crear una función "callback".

Una función callback es ejecutada después de que el efecto actual finaliza.

La sintaxis típica es: $(selector).hide(speed,callback);

El ejemplo siguiente tiene un parámetro callback que es una función que será ejecutada luego de que el efecto "hide" es completado:

Ejemplo con Callback


  $("button").click(function(){
    $("p").hide("slow", function(){
      alert("The paragraph is now hidden");
    });
  });

El ejemplo siguiente no tiene parámetro callback y el mensaje de alerta será mostrado antes de que el efecto "hide" sea completado:

Ejemplo sin Callback


  $("button").click(function(){
    $("p").hide(1000);
    alert("The paragraph is now hidden");
  });
