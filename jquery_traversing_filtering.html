jQuery Traversing - Filtering

Los métodos first(), last(), eq(), filter() y not()

Los métodos de filtrado más básicos son first(), last() y eq(), los cuales nos permiten seleccionar un elemento específico en base a su posición dentro de un grupo de elementos.

Otros métodos de filtrado, como filter() y not() nos permiten seleccionar elementos que coincidan o no coincidadn con un determinado criterio.

El método jQuery first()

El método first() devuelve el primer elemento del tipo de elemento especificado.

El siguiente ejemplo selecciona el primer elemento <div>:

Ejemplo


  $(document).ready(function(){
    $("div").first();
  });

El método jQuery last()

El método last() devuelve el último elemento del tipo de elemento especificado.

El ejemplo que sigue selecciona el último elemento <div>:

Ejemplo


  $(document).ready(function(){
    $("div").last();
  });

El método jQuery eq()

El método eq() devuelve un elemento con un número específico de índice en los elementos seleccionados.

El número de índice comienza con 0, así que el primer elemento tendrá el número de índice 0 y no 1. El ejemplo siguiente selecciona el segundo elemento <p> (índice número 1):

Ejemplo


  $(document).ready(function(){
    $("p").eq(1);
  });

El método jQuery filter()

El método filter() nos deja especificar un criterio. Los elementos que no coinciden con el criterio son quitados de la selección, y aquellos que coinciden son devueltos.

El ejemplo siguiente devuelve todos los elementos <p> con nombre de clase "intro":

Ejemplo


  $(document).ready(function(){
    $("p").filter(".intro");
  });

El método jQuery not()

El método not() devuelve todos los elementos que no coinciden con un criterio.

Tip: el método not() es el método opuesto de filter().

El siguiente ejemplo devuelve todos los elementos <p> que no tienen el nombre de clase "intro":

Ejemplo


  $(document).ready(function(){
    $("p").not(".intro");
  });
