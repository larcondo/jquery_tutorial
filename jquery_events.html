jQuery Event Methods

jQuery está hecho a medida para responder a eventos en una página HTML.

¿Qué son los Eventos?

Todas las diferentes acciones del usuario a las que una página puede responder son llamadas eventos.

Un evento representa el preciso momento cuando algo ocurre.

Ejemplos:

  • Mover el mouse sobre un elemento
  • Seleccionar un radio button
  • Hacer click en un elemento

    • Aquí van algunos ejemplos de eventos del DOM:

    Mouse Events Keyboard Events Form Events Document/Window Events
    click keypress submit load
    dblclick keydown change resize
    mouseenter keyup focus scroll
    mouseleave blur unload

    Sintaxis jQuery para métodos de Eventos

    En jQuery, la mayoría de los eventos del DOM tienen su método jQuery equivalente.

    Para asignar un click event a todos los párrafos de una página se debe hacer lo siguiente:

    
  $("p").click();

    El próximo paso es definir qué debería pasar cuando el evento ocurra. Debemos pasarle una función al evento:

    
  $("p").click(function(){
    // Las acciones van aquí!
  });

    jQuery Event Methods comunmente usados

    $(document).ready()

    El método $(document).ready() nos permite ejecutar una función cuando el documento está totalmente cargado. Este método ya fue explicado previamente.

    click()

    El método click() fija una función para manejar el evento de un elemento HTML.

    La función se ejecuta cuando el ususario hace click en un elemento HTML.

    
  $("p").click(function(){
    $(this).hide();
  });

    dblclick()

    La función se ejecuta cuando el usuario hace doble click en el elemento HTML.

    
  $("p").dblclick(function(){
    $(this).hide();
  });

    mouseenter()

    La función se ejecuta cuando el puntero del mouse ingresa en el elemento HTML:

    
  $("#p1").mouseenter(function(){
    alert("You entered p1!");
  });

    mouseleave()

    La función se ejecuta cuando el puntero del mouse deja el elemento HTML:

    
  $("#p1").mouseleave(function(){
    alert("Bye! You now leave p1!");
  });

    mousedown

    La función se ejecuta cuando el botón derecho, izquierdo o del medio del mouse es presionado mientras el mouse está sobre el elemento HTML:

    
  $("#p1").mousedown(function(){
    alert("Mouse down over p1!");
  });

    mouseup()

    La función se ejecuta cuando el botón derecho, izquierdo o del medio del mouse es soltado mientras el mouse está sobre el elemento HTML:

    
  $("#p1").mouseup(function(){
    alert("Mouse up over p1!");
  });

    hover()

    El método hover() toma dos funciones y es una combinación de los métodos mouseenter() y mouseleave().

    La primera función se ejecuta cuando el mouse ingresa al elemento HTML y la segunda cuando el mouse sale del elemento HTML:

    
  $("#p1").hover(function(){
    alert("You entered p1!");
  },
  function(){
    alert("Bye! You now leave p1!");
  });

    focus()

    La función se ejecuta cuando el campo del formulario obtiene el foco:

    
  $("input").focus(function(){
    $(this).css("background-color", "#cccccc");
  });

    blur()

    La función se ejecuta cuando el campo del formulario pierde el foco:

    
  $("input").blur(function(){
    $(this).css("background-color", "#ffffff");
  });

    El método on()

    El método on() fija uno o más manejadores de eventos a los elementos seleccionados.

    Agregar un evento click a un elemento <p>:

    
  $("p").on("click", function(){
    $(this).hide();
  });

    Agregar múltiples manejadores de eventos a un elemento <p>:

    
  $("p").on({
    mouseenter: function(){
      $(this).css("background-color", "lightgray");
    },
    mouseleave: function(){
      $(this).css("background-color", "lightblue");
    },
    click: function(){
      $(this).css("background-color", "yellow");
    }
  });
