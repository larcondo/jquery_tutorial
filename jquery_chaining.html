jQuery - Encadenamiento

Con jQuery podemos encadenar juntos acciones/métodos.

Encadenar nos permite correr multiples métodos jQuery (en el mismo elemento) en una sola línea de código.

El método jQuery Encadenar

Hasta ahora hemos escrito un declaración jQuery a la vez (una después de otra).

Sin embargo, hay una técnica llamada "chaining" que nos permite ejecutar múltiples comandos jQuery, una después de otra, sobre el mismo elemento.

Tip: De esta manera, el navegador no tiene que encontrar el mismo elemento más de una vez.

Para encadenar una acción, simplemente añadimos la acción a la acción previa.

El ejemplo siguiente encadena los métodos css(), slideUp() y slideDown(). El elemento "p1" primero cambia a rojo, luego se desliza hacia arriba y posteriormente se desliza hacia abajo:

Ejemplo


  $("#p1").css("color", "red").slideUp(2000).slideDown(2000);

Incluso podríamos haber agregado más métodos si fuese necesario.

Tip: Cuando encadenamos, la línea de código podría volverse bastante larga. Sin embargo, jQuery no es tan estricto en la sintaxis. Podemos darle el formato que queramos, incluyendo saltos de línea e indentaciones.

Escribiendo el código de la siguiente forma también funcionaría correctamente:

Ejemplo


  $("#p1").css("color", "red")
    .slideUp(2000)
    .slideDown(2000);

jQuery elimina los espacios en blanco extras y ejecuta las líneas anteriormente mostradas como una sola línea larga de código.
