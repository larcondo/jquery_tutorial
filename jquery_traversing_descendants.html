jQuery Traversing - Descendants

Con jQuery podemos recorrer hacia abajo el árbol DOM para encontrar descendientes de un elemento.

Un descendiente es un hijo, un nieto, un bis-nieto, y así sucesivamente.

Recorrer hacia abajo el árbol DOM

Dos métodos jQuery útiles para recorrer hacia abajo el árbol DOM son:

  • children()
  • find()

    • El método jQuery children()

    El método children() retorna todos los hijos directos del elemento seleccionado.

    Este método solo recorre un solo nivel hacia abajo en el árbol DOM.

    El siguiente ejemplo retorna todos los elementos que son hijos directos de cada elemento <div>:

    Ejemplo

    
  $(document).ready(function(){
    $("div").children();
  });

    También podemos utilizar un parámetro opcional para filtrar la búsqueda del hijo.

    El ejemplo siguiente retorna todos los elementos <p> con el nombre de clase "first", que son hijos directos de un elemento <div>:

    Ejemplo

    
  $(document).ready(function(){
    $("div").children("p.first");
  });

    El método jQuery find()

    El método find() retorna elementos descendientes del elemento seleccionado hacia abajo hasta el último descendiente.

    El ejemplo siguiente retorna todos los elementos <span> que son descendientes del elemento <div>:

    Ejemplo

    
  $(document).ready(function(){
    $("div").find("span");
  });

    El siguiente ejemplo retorna todos los descendientes del <div>:

    Ejemplo

    
  $(document).ready(function(){
    $("div").find("*");
  });
