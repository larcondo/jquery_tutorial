Selectores jQuery

Los selectores jQuery son una de las partes más importantes de la librería jQuery.

Selectores jQuery

Los selectores jQuery permiten seleccionar y manipular elementos HTML.

Son utilizados para "buscar" (o seleccionar) elementos HTML en base a su nombre, id, clase, tipo, atributos, valors de atributos y mucho más. Está basado en los Selectores CSS existentes e incluso, tienen sus propios selectores.

Todos los selectores en jQuery comienzan con el signo de dolar y paréntesis: $().

El selector element

Selecciona elementos en base a el nombre del elemento.

Podemos seleccionar todos los elementos <p> en una página así:


  $("p")

El selector #id

El selector #id de jQuery utiliza el atributo id de un tag HTML para encontrar un elemento específico.

Un id debe ser único dentro de una página, así que debemos utilizar el selector #id cuando queremos encontrar un solo y único elemento.

Para encontrar un elemento con un id específico, se escribe el caracter "hash" (#) seguido por el id del elemento HTML:


  $("#test")

El selector .class

El selector .class de jQuery encuentra elementos de una clase específica.

Para encontrar elementos con una clase específica, escribimos el caracter "period" (.) seguido del nombre de la clase:


  $(".test")

Más ejemplos de Selectores jQuery

Sintaxis Descripción
$("*") Selecciona todos los elementos
$(this) Selecciona el elemento HTML actual
$("p.intro") Selecciona todos los elementos <p> con class="intro"
$("p:first") Selecciona el primer elemento <p>
$("ul li:first") Selecciona el primer elemento <li> del primer <ul>
$("ul li:first-child") Selecciona el primer elemento <li> de cada <ul>
$("[href]") Selecciona todos los elementos con un atributo "href"
$("a[target='_blank']") Selecciona todos los elementos <a> con un valor de target attribute igual a "_blank"
$("a[target!='_blank']") Selecciona todos los elementos <a> con un valor de target attribute NO igual a "_blank"
$(":button") Selecciona todos los elementos <button> y los elementos <input> de type="button"
$("tr:even") Selecciona todos los elementos <tr> pares
$("tr:odd") Selecciona todos los elementos <tr> impares

Funciones en Archivos Separados

Si un sitio web tiene muchas páginas y queremos que nuestras funciones jQuery sean fáciles de mantener, podemos poner dichas funciones en archivos .js separados.

Cuando presentamos jQuery, las funciones fueron agregadas directamente en la sección <head>. Sin embargo, a veces es preferible ponerlas en archivos separados, como en el ejemplo siguiente: (se utiliza el atributo src para referir al archivo .js)


  <head>
  <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.6.3/jquery.min.js"></script>
  <script src="my_jquery_functions.js"></script>
  </head>
