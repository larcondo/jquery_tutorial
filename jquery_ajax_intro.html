jQuery - Introducción AJAX

AJAX es el arte de intercambiar información/datos con un servidor y actualizar partes de una página web sin tener que recargar toda la página.

¿Qué es AJAX?

AJAX = Asynchronous JavaScript and XML.

En pocas palabras, AJAX se trata de cargar datos en el fondo y mostrarlos en la página web sin recargar toda la página nuevamente.

Ejemplos de aplicacions que utilizan AJAX son: Gmail, Google Maps, Youtube y Facebook tabs.

¿Qué ocurre con jQuery y AJAX?

jQuery provee varios métodos para la funcionalidad AJAX.

Con los métodos jQuery AJAX podemos solicitar texto, HTML, XML o JSON de un servidor remoto usando HTTP Get and HTTP Post y podremos cargar datos externos directamente en los elementos HTML seleccionados de nuestra página web!

Sin jQuery, codificar AJAX puede ser un poco complicado!
Escribir código AJAX normal puede ser un poco complicado ya que los diferentes navegadores tienen diferentes sintaxis para la implementación AJAX. Esto significa que tendremos que escribir código extra para testear para diferentes navegadores. Sin embargo, el equipo jQuery se a preocupado de esto para nosotros, así que han escrito funcionalidad AJAX con una sola línea de código.
