Sintaxis jQuery

Con jQuery podemos seleccionar (query) elementos HTML y realizar "acciones" sobre ellos.

La sintaxis básica es: $(selector).action()

  • Un signo $ para definir/acceder a jQuery
  • Un (selector) "consultar (o buscar)" elementos HTML
  • Una action() jQuery a ser ejecutada en los elementos HTML.

    • Ejemplo:

  • $(this).hide() - oculta el elemento actual.
  • $("p").hide() - oculta todos los elementos <p>.
  • $(".test").hide() - oculta todos los elementos con la clase class="test".
  • $("#test").hide() - oculta todos los elementso con el id id="test".
    • Nota: jQuery utiliza la sintaxis CSS para seleccionar elementos.

    El evento Document Ready

    Notarás que todos los métodos utilizados en los ejemplos estarán "dentro" del evento document ready:

    
  $(document).ready(function(){

    // jQuery methods go here...

  });

    Esto es para prevenir que cualquier código jQuery se ejecute antes de que el documento termine de cargar (esté listo, "is ready").

    Es una buena práctica esperar a que el documento esté totalmente cargado y listo antes de trabajar con el mismo. Esto también te permite tener el código JavaScript antes del body del documento, en la sección head.

    Aquí van algunos ejemplos de acciones que pueden fallar si los métodos se ejecutan antes de que el documento esté totalmente cargado:

  • Tratar de ocultar un elemento que no ha sido creado aún.
  • Tratar de obtener el tamaño de una imagen que todavía no ha sido cargada aún.

    • Tip: El jQuery team también ha creado un método abreviado para el vento document ready:

    
  $(function(){

    // jQuery methods go here...
  
  });
