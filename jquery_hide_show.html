Efectos jQuery - Hide y Show

jQuery hide() y show()

Con jQuery podemos ocultar y mostrar elementos HTML con los métodos hide() y show():


  $("#hide").click(function(){
    $("p").hide();
  });
  
  $("#show").click(function(){
    $("p").show();
  });

Sintaxis:


  $(selector).hide(speed,callback);
  $(selector).show(speed,callback);

El parámetro opcional speed especifica la velocidad en la que se oculta/muestra el elemento y puede tomar los siguientes valores: "slow", "fast" o milisegundos.

El parámetro opcional callback es una función a ser ejecutada luego de que el método hide() o show() se complete.

El ejemplo siguiente muestra el uso del parámetro speed con el método hide():


  $("button").click(function() {
    $("p").hide(1000);
  });

jQuery toggle()

También podemos alternar (toggle) entre mostrar y ocultar un elemento con el método toggle().

Los elementos que se están mostrando son ocultados y los que están ocultos son mostrados:


  $("button").click(function(){
    $("p").togle();
  });

Sintaxis:


  $(selector).toggle(speed,callback);

El parámetro opcional speed puede tomar los siguientes valores: "slow", "fast" o milisegundos.

El parámetro opcional callback es una función a ser ejecutada luego de que toggle() finalice.
