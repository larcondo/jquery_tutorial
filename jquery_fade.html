jQuery Fading - Métodos

Con jQuery podemos desvanecer (fade) un elemento mostrándolo u ocultándolo.

jQuery tiene los siguientes métodos para desvanecer:

  • fadeIn()
  • fadeOut()
  • fadeToggle()
  • fadeTo()

    • El método jQuery fadeIn()

    El método jQuery fadeIn() es utilizado para mostrar gradualmente un elemento oculto:

    Sintaxis:

    
  $(selector).fadeIn(speed,callback);

    El parámetro opcional speed especifica la duración del efecto. Puede tomar los siguientes valores: "slow", "fast" o milisegundos.

    El parámetro opcional callback es una función a ser ejecutada luego de que el método fadeIn() se complete.

    El ejemplo siguiente muestra el método fadeIn() con diferentes parámetros:

    
  $("button").click(function(){
    $("#div1").fadeIn();
    $("#div2").fadeIn("slow");
    $("#div3").fadeIn(3000);
  });

    El método jQuery fadeOut()

    El método jQuery fadeOut() es utilizado para ocultar gradualmente un elemento visible.

    Sintaxis:

    
  $(selector).fadeOut(speed,callback);

    El parámetro opcional speed especifica la duración del efecto. Puede tomar los siguientes valores: "slow", "fast" o milisegundos.

    El parámetro opcional callback es una función a ser ejecutada luego de que el método fadeOut() se complete.

    El ejemplo siguiente muestra el método fadeOut() con diferentes parámetros:

    
  $("button").click(function(){
    $("#div1").fadeOut();
    $("#div2").fadeOut("slow");
    $("#div3").fadeOut(3000);
  });

    El método jQuery fadeToggle()

    El método jQuery fadeToggle() alterna entre los métodos fadeIn() y fadeOut().

    Si el elemento está oculto, fadeToggle() lo mostrará gradualmente.

    Si el elemento está visible, fadeToggle() lo ocultará gradualmente.

    Sintaxis:

    
  $(selector).fadeToggle(speed,callback);

    El parámetro opcional speed especifica la duración del efecto. Puede tomar los siguientes valores: "slow", "fast" o milisegundos.

    El parámetro opcional callback es una función a ser ejecutada luego de que el método fadeToggle() se complete.

    El ejemplo siguiente muestra el método fadeToggle() con diferentes parámetros:

    
  $("button").click(function(){
    $("#div1").fadeToggle();
    $("#div2").fadeToggle("slow");
    $("#div3").fadeToggle(3000);
  });

    El método jQuery fadeTo()

    El método jQuery fadeTo() permite mostrar/ocultar gradualmente a una dada opacidad (valor entre 0 y 1).

    Sintaxis:

    
  $(selector).fadeTo(speed,opacity,callback);

    El parámetro opcional speed especifica la duración del efecto. Puede tomar los siguientes valores: "slow", "fast" o milisegundos.

    El parámetro obligatorio opacity especifica a qué valor de opacidad se mostrará/ocultará gradualmente (valor entre 0 y 1).

    El parámetro opcional callback es una función a ser ejecutada luego de que el método fadeTo() se complete.

    El ejemplo siguiente muestra el método fadeTo() con diferentes parámetros:

    
  $("button").click(function(){
    $("#div1").fadeTo("slow", 0.15);
    $("#div2").fadeTo("slow", 0.4);
    $("#div3").fadeTo("slow", 0.7);
  });
